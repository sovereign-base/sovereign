<objective>
Add a first-class `upgrade` command to the `sovereign-cli` launcher that moves an EXISTING SOVEREIGN install to the newest packaged version while preserving the user's `.sovereign/` content.

Purpose: Users today only have `init`, which fresh-installs anywhere. `upgrade` adds a clearer, safer affordance: it refuses to run where nothing is installed (steering the user to `init`), and otherwise reuses the already-correct, non-destructive `runInstall` core (re-copies skills/agents/engine/references, re-stamps VERSION, leaves `.sovereign/` untouched). No install logic is reinvented.

Output:
- `runUpgrade(opts)` exported from `engine/bin/lib/install.cjs`
- `upgrade` (alias `update`) command + `renderUpgrade()` in `engine/bin/sovereign.cjs`, plus a `usage()` line
- `engine/test/upgrade.test.cjs` covering the not_installed / updated / up_to_date paths and the launcher UX
- A short "Upgrading" note in `README.md`
</objective>

       ```js
       /**
        * @typedef {InstallResult | {
        *   ok: false,
        *   status: 'not_installed',

        *   installed_version: string
        * }} UpgradeResult

       /**
        * Upgrade an EXISTING SOVEREIGN install to the packaged version. Unlike init,
        * this refuses to fresh-install: if there is no `.sovereign/.sovereign-version`
        * stamp it does nothing and reports `not_installed`. When a stamp is present it
        * delegates wholesale to runInstall — which is already non-destructive (preserves
        * `.sovereign/`), re-copies skills/agents/engine, and re-stamps VERSION.
        * @param {Object} opts
        * @param {string} opts.cwd
        * @param {'project'|'global'} opts.target
        * @param {'quick'|'full'} opts.mode
        * @param {string} opts.packageRoot
        * @returns {UpgradeResult}
        */
       function runUpgrade(opts) {
         const stampPath = path.join(opts.cwd, '.sovereign', '.sovereign-version');
         const stamp = (safeReadFile(stampPath) || '').trim();
         if (!stamp) {
           return {
             ok: false,
             status: 'not_installed',
             target: opts.target,
             installed_version: (safeReadFile(path.join(opts.packageRoot, 'VERSION')) || '').trim(),
           };
         }
         return runInstall(opts);
       }
       ```

       ```js
       function renderUpgrade(r, dir) {

           say('  Run `npx sovereign-cli init` first.');

           say('    Your .sovereign/ content was preserved.');
       ```
