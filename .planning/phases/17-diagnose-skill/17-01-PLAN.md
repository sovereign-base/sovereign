       - Orient (one call): the verbatim orient snippet from <authoring_facts>. Then parse `config.commit_docs` + `context_injection.glossary_path`; DETECT the project's own test/run command — never hardcode (mirror tdd's exact agnostic phrasing).

       - Step 5 — Verify: re-run the repro/failing test (now green) AND the project's full SUITE (no REGRESSION). Then run **`/sentinel`** for the standards pass on the fix.
